# Python Graph

## Functional programming with graphs in Python


### Dependencies - Make sure to install Go Task


#### Windows

----

#### Using Chocolatey

```sh
choco install task
```

#### Using Winget

```sh
winget install Task.Task
```

#### Mac

----

```sh
brew install go-task/tap/go-task
```

#### Linux


```sh
sudo apt install task
```


### Run the app

```sh
task run
```


### Start Tailwind watcher

```sh
task build
```
